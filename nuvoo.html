Project Overview

A design-driven social innovation project that reimagines how unused but functional eyeglasses can be repurposed in South Korea.

Problem Statement

Many households in South Korea accumulate old but functional glasses, with no clear or convenient way to donate them. This leads to unnecessary waste and missed opportunities for social good, as these glasses could benefit those in need.

Solution Approach

NUVOO introduces a smart donation system using accessible kiosks and local partnerships. The system makes it easy for anyone to donate glasses, ensures efficient collection and distribution, and promotes sustainability by giving unused items a second life.

Design Process

  • Conducted user research to identify the problem of unused glasses and lack of donation channels
  • Developed service design strategy for accessible donation system
  • Created user-friendly kiosk interface design for easy drop-off and tracking
  • Established comprehensive branding and visual identity system for NUVOO
  • Built partnerships framework for local collection and distribution network
  • Prototyped 3D models and interactive interfaces using Figma and Shapr3D
