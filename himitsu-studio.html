Himitsu Studio
Personal Project  |  Interactive Web Application  |  2025
Tools: React, Next.js, TypeScript, GSAP, Tailwind CSS
Role: Full-Stack Developer & Designer

Overview

Himitsu Studio (秘密スタジオ) is an interactive art therapy web application where users can express their hidden feelings through digital origami. The platform provides a safe, creative space for emotional expression through a unique desk-drawer interface that reveals a secret studio.

Himitsu Studio Interface Preview
Interactive desk interface with sliding drawer animation

Technical Implementation

Built with modern web technologies to create smooth, engaging interactions:

  • React 18 + Next.js 14: Component-based architecture with server-side rendering
  • TypeScript: Type-safe development for better code quality
  • GSAP: Professional-grade animations for the drawer sliding effect
  • Tailwind CSS: Utility-first styling for responsive design
  • State Management: React hooks for managing user journey flow
React Next.js TypeScript GSAP Tailwind CSS Responsive Design

Key Features

  • Interactive Desk Interface: Click-to-open drawer with realistic sliding animation
  • Messy Drawer Discovery: Hidden contents reveal the journey to the secret studio
  • Origami Selection: Choose from different paper types and colors
  • Emotional Expression: Write feelings and watch them transform into origami art
  • Folding Animation: Smooth GSAP-powered origami folding sequence
  • Gallery System: Save and view created origami pieces
  • Mobile Responsive: Optimized experience across all devices

Design Process

The concept was inspired by the idea of hidden emotions and the therapeutic nature of creative expression. The desk-drawer metaphor represents how we often hide our true feelings, and the messy drawer symbolizes the chaos of emotions that can be transformed into something beautiful.

Key design decisions:

  • Hand-drawn style illustrations for a personal, organic feel
  • Warm color palette (ambers and browns) to create comfort
  • Realistic physics in animations to enhance believability
  • Progressive disclosure to guide users through the experience

Technical Challenges

Several interesting technical challenges were solved during development:

  • Layering System: Proper z-index management for drawer sliding behind desk frame
  • Animation Synchronization: Coordinating multiple GSAP timelines for smooth transitions
  • State Management: Complex state machine for managing user journey flow
  • Image Optimization: Next.js Image component for performance
  • Responsive Animations: Ensuring smooth animations across different screen sizes

Impact & Learning

This project demonstrates the intersection of technology and emotional well-being, showing how interactive design can create meaningful experiences. It showcases advanced frontend development skills while addressing real human needs for creative expression and emotional processing.

Key learnings:

  • Advanced GSAP animation techniques and timeline management
  • Complex state management patterns in React
  • Performance optimization for animation-heavy applications
  • User experience design for emotional and therapeutic contexts
